#### ETW v2.0

**propName回调第二个参数PTRACE_EVENT_INFO info，用来标识具体事件的Event，如ProcessId/UserSID等：**

```c
	for(..........)
	{
		auto& pi = info->EventPropertyInfoArray[i];
		propName = (PCWSTR)((BYTE*)info + pi.NameOffset);
	}
```

**具体的事件字段含义，See Msnd: **
```
	https://learn.microsoft.com/zh-cn/windows/win32/etw/msnt-systemtrace
```
